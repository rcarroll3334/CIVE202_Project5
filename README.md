# Project Overview
This project analyzes Total Coliform Rule (TCR) violations in public water systems using EPA SDWIS data from 2021–2025. The goal of the project is to identify trends, compare violations across states and system sizes, and visualize relationships within the dataset using Python.

The project focuses on:
- Data cleaning and preprocessing
- Trend analysis
- State comparisons
- System size comparisons
- Data visualization

---

# Objectives
The objectives of this project were to:

- Combine quarterly SDWIS datasets from 2021–2025
- Clean and prepare the datasets for analysis
- Analyze TCR violations across selected states
- Compare violations by population category (system size)
- Evaluate trends over time
- Create visualizations to communicate findings clearly

---

# Dataset Information
The data used in this project was obtained from the EPA Safe Drinking Water Information System (SDWIS).

### Dataset Includes
- Public Water System (PWS) information
- Population served
- Violation type
- Compliance dates
- State information
- System size categories

### Time Range
2021 Q1 – 2025 Q4
